# Refinement Template

Use this Jira Markup shape for refined Spike tickets. Preserve useful existing content when refining an existing ticket, improve unclear parts, and omit optional sections when empty.

```jira
h2. Title

h2. Research Question

h2. Context

h2. Scope

h2. Expected Deliverable

h2. Acceptance Criteria

h2. Implementation Notes

h2. Test Notes

h2. Open Questions

h2. Assumptions and TBD
```
