# Recipes

Recipes are playbooks for combining agents, skills, commands, and human judgment.

## Add a recipe when

- The workflow spans more than one tool.
- The order of steps matters.
- A future you should not have to rediscover the process.

## Recipe contract

Each recipe should include:

- when to use it
- tools involved
- step-by-step flow
- expected output
- validation checklist
