# Primary Agents

Primary agents coordinate workflows. They may delegate, sequence work, and synthesize results.

## Add a primary agent when

- The workflow has multiple phases.
- It needs to choose between subagents or skills.
- It owns orchestration, not one narrow task.

## Contract

- State what it coordinates.
- State what it must not do directly.
- Name the subagents or skills it may call.
- Define the final output users should expect.
