# Agent Artifacts

This context names reusable agent artifacts and interaction patterns used in this repository.

## Language

**Rubber Ducking**:
A guided thinking interaction where the agent helps the user explain a confusing problem, surface assumptions, form hypotheses, and choose the next small experiment without taking over the solution.
_Avoid_: Rubber duck, coach, consultant, grill session

**Thinking Command**:
An intentional command entrypoint for guided thinking modes, including rubber ducking, plan grilling, and documentation-aware grilling.
_Avoid_: Rubber ducking command, generic help command
